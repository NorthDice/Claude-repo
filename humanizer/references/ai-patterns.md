# AI Patterns to Remove

## Opening clichés — always delete

These phrases signal AI immediately. Cut them entirely:

**English:**
- "Certainly!", "Absolutely!", "Of course!", "Sure!"
- "Great question!", "That's a great point!"
- "I'd be happy to help"
- "In today's world / In today's fast-paced world"
- "It's important to note that..."
- "It's worth mentioning that..."
- "As an AI language model..."
- "I hope this helps!"
- "Feel free to reach out"

**Russian:**
- «Конечно!», «Разумеется!», «Безусловно!»
- «Отличный вопрос!»
- «В современном мире...»
- «Важно отметить, что...»
- «Следует упомянуть, что...»
- «Надеюсь, это помогло!»
- «Не стесняйтесь обращаться»

## Transition bloat — simplify or cut

These words are overused by AI. Replace with something simpler or remove:

| AI word | Human alternative |
|---------|------------------|
| Furthermore | Also / And |
| Moreover | On top of that / Also |
| Additionally | And |
| Subsequently | Then / After that |
| Nevertheless | Still / But |
| Consequently | So |
| Utilize | Use |
| Leverage | Use |
| Facilitate | Help / Make easier |
| Implement | Do / Use / Add |
| Optimize | Improve / Fix |
| Comprehensive | Full / Complete |
| Robust | Strong / Solid |
| Seamless | Smooth / Easy |

**Russian equivalents to cut:**
- «Таким образом» (в начале каждого абзаца) → убрать
- «В заключение» → убрать или «Итого»
- «Следовательно» → «Значит» / «Поэтому»
- «Осуществлять» → «делать»
- «Использовать в полной мере» → «использовать»

## Over-structuring — flatten

AI loves excessive bullet points and headers. Flatten when content is simple:

```
// AI output
**Key benefits:**
- It saves time
- It reduces errors
- It improves quality

// Human version
It saves time, cuts errors, and the quality goes up.
```

## Conclusion paragraphs — cut or shorten

AI always wraps up with a summary paragraph. Usually redundant:
- "In conclusion, we have explored..."
- "To summarize the key points..."
- «Таким образом, мы рассмотрели...»
- «Подводя итоги, можно отметить...»

Cut entirely or replace with one short sentence if needed.

## Passive voice overuse

AI defaults to passive. Switch to active:
```
// AI
"The solution was implemented by the team"
// Human
"The team implemented the solution" (or just "We implemented it")

// AI (RU)
«Задача была выполнена»
// Human
«Мы выполнили задачу» / «Задача выполнена»
```
