# Human Writing Techniques

## Sentence length variation

Humans don't write in uniform sentence lengths. Mix short and long:

```
// AI — all sentences similar length, monotone
This approach has several advantages. It reduces the complexity of the system.
It also improves maintainability. Furthermore, it enables better scalability.

// Human — varied rhythm
This approach has a few real advantages. It cuts complexity. And down the line,
the system is much easier to maintain and scale.
```

## Contractions (English)

Use in casual and neutral registers:
- it is → it's
- do not → don't
- cannot → can't
- they are → they're
- we will → we'll
- I would → I'd

## Direct opinions and hedges

Humans have opinions and sometimes admit uncertainty:
```
// AI — neutral, hedged everywhere
"This approach may potentially offer some benefits in certain scenarios."

// Human — direct
"This works well. There are edge cases, but for most situations it's fine."
```

## Filler words (use sparingly)

A few make text feel alive. Don't overdo:
- "basically", "honestly", "actually", "just", "really"
- RU: «по сути», «честно говоря», «в общем», «просто»

## Informal connectors

Start sentences with these when it feels natural:
- "And...", "But...", "So...", "Look,..."
- RU: «И...», «Но...», «Так что...», «Слушай,...»

## Cut redundancy

AI repeats itself. Remove sentences that say what was already said:
```
// AI
"The system is fast. Performance is high. It responds quickly."

// Human
"The system is fast."
```

## Specific over generic

Humans are concrete, AI is abstract:
```
// AI
"This solution provides significant improvements to overall system performance."

// Human
"This cut response time from 400ms to 80ms."
```

## Russian-specific techniques

- Avoid «данный» / «данная» — use «этот» / «эта»
- Avoid «осуществлять», «производить», «обеспечивать» — use simple verbs
- Avoid «в рамках» when «в» or «для» works
- Use incomplete sentences where natural: «Хорошая идея. Попробуем.»
- Vary paragraph length — one-sentence paragraphs are fine
