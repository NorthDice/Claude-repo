# Claude Skills

A collection of custom skills for Claude Code.

## Skills

- **go-code-reviewer** — Review Go code for correctness, idiomatic style, concurrency issues, and performance
- **humanizer** — Rewrite AI-generated text to sound human
- **learnings-cleaner** — Clean and compress `learnings.md` files across all skills
- **meta-skill-creator** — Create and refactor Claude Code skills following best practices

## Usage

Copy any skill folder into `~/.claude/skills/` and it will be available in Claude Code automatically.
